# 시도시군구코드

대한민국 행정구역 코드 참조표. 교육통계 데이터의 `시도명` 필터링 및 지역별 집계 시 코드 매핑 기준으로 사용한다.

---

## 코드 구조

| 필드 | 자릿수 | 설명 |
|------|--------|------|
| `시도코드` | 2자리 | 광역 시·도 코드 |
| `시군구코드` | 5자리 | 기초 시·군·구 코드 (앞 2자리 = 시도코드) |

> `전체 / 00 / 전체 / 00000` 은 전국 합계를 의미하는 특수 코드다.

## DuckDB 활용 예시

```sql
-- 서울특별시 학교만 필터링
SELECT * FROM read_parquet('data/raw_parquets/0001*.parquet')
WHERE 시도명 = '서울특별시'

-- 시도코드로 JOIN (교육통계 데이터에 시도명이 있으므로 시도명 직접 사용 권장)
SELECT a.시도명, COUNT(*) as 학교수
FROM read_parquet('data/raw_parquets/0001*.parquet') a
GROUP BY a.시도명
ORDER BY 학교수 DESC
```
