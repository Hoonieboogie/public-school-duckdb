## 001. 공통\_학교속성

**파일명**: `0001. 공통_학교속성(09-23)(100%)_지역정보미제공.csv`
**행 수**: 313,576행 | **컬럼 수**: 30개
**파일 크기**: CSV 80MB → Parquet 1.8MB (45배 압축)

### 개요
전국 모든 학교의 **기본 속성 정보**를 담은 마스터 참조 테이블이다.
학교 유형, 설립 형태, 운영 방식 등 학교를 분류하는 범주형 정보가 중심이다.
파일명의 `_지역정보미제공`은 개인정보 보호를 위해 읍·면·동 단위 이하 주소가 제외됨을 의미한다.

### 컬럼 명세

| 컬럼명 | 설명 | 분석 활용 예시 |
|--------|------|---------------|
| `학교급명` | 학교급 (유치원/초등학교/중학교/고등학교/특수학교) | 학교급별 필터링 |
| `시도명` | 광역 시·도명 | 지역별 집계 |
| `시군명` | 시·군·구명 | 세부 지역별 분석 |
| `학제명` | 학교제도명 | 학제 구분 필터 |
| `설립구분명` | 국립 / 공립 / 사립 | 설립 유형별 비교 |
| `종교단체명` | 종교 재단 소속 여부 및 종류 | 사립학교 특성 파악 |
| `운영방식명` | 학교 운영 방식 구분 | 운영 형태 분류 |

| `법인종류명` | 학교법인 종류 | 사립학교 법인 분석 |
| `설립주체명` | 설립 주체 (국가/지자체/법인/개인 등) | 공사립 세분류 |
| `설립유형명` | 설립 유형 세부 구분 | 설립 배경 분석 |

| `본교분교구분명` | 본교 / 분교 구분 | 소규모 분교 식별 |
| `교장공모제여부` | 교장 공모제 적용 여부 (Y/N) | 교육 정책 분석 |
| `학교설립일자` | 학교 개교일자 | 학교 역사 분석 |
| `학제유형명` | 학제 유형 세부 구분 | 학제 세분류 |

| `남녀공학구분명` | 남학교 / 여학교 / 남녀공학 구분 | 성별 학교 분포 분석 |
| `학교수` | 해당 조건의 학교 수 | 집계 기준 지표 |

### 주요 분석 활용
- 전국 학교 지도 시각화 (ROADMAP Step 1)
- 공립/사립 비율 지역별 비교
- 특수지(도서벽지) 학교 식별
- 고교 유형(일반고/특목고/자율고) 분포 분석

## 파일 간 JOIN 예시

```python
import polars as pl

# 학교속성 + 학생현황 JOIN
attrs   = pl.scan_parquet("parquets/0001*.parquet")
student = pl.scan_parquet("parquets/0004*.parquet")

result = (
    attrs
    .join(student, on=["학교ID", "조사년도"], how="left")
    .filter(pl.col("조사년도") == 2023)
    .filter(pl.col("시도명") == "서울특별시")
    .select(["학교ID", "학교급명", "설립구분명", "유초중등학생_학생수"])
    .collect()
)
```

```sql
-- DuckDB: 학교급별 연도별 학생 수 추이
SELECT
    a.조사년도,
    a.학교급명,
    SUM(s.유초중등학생_학생수) AS 전체학생수
FROM read_parquet('parquets/0001*.parquet') a
JOIN read_parquet('parquets/0004*.parquet') s
  ON a.학교ID = s.학교ID AND a.조사년도 = s.조사년도
GROUP BY a.조사년도, a.학교급명
ORDER BY a.조사년도, a.학교급명
```
